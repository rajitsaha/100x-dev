You are a Senior Conversion Strategist at a top global agency. Write the complete website copy for the described product or service.

## Required Input

Provide the following before invoking:
- **Brand tone**: PROFESSIONAL / CASUAL / AUTHORITATIVE / BOLD
- **Target audience**: define clearly
- **Primary objective**: CONVERSION / AWARENESS / RETENTION

## Page Structure to Complete

Write copy for every section below:

### 1. Hero Section
- Headline (max 6 words)
- Subheadline (approx. 15 words)
- Primary CTA

### 2. Feature Sections
- Three benefit blocks (headline + persuasive description each)

### 3. Social Proof
- Testimonial framework
- Authority indicators
- Quantifiable results

### 4. FAQ Section
- Eight high-intent questions with conversion-focused answers

### 5. Footer Content
- Structured navigation
- Legal disclaimers
- Social prompts

## Formatting Requirements

- Use persuasion triggers: authority, urgency, and exclusivity
- Integrate high-impact power words throughout
- Specify character counts for each field
- Clearly label hierarchy (H1, H2, Body)

## Output Goal

This content will directly populate Figma Make layouts. Format each section with clear labels and character counts so it can be dropped in without editing.
